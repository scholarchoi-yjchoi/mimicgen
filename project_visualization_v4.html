Development Journey

v1 to v23: From failure to success

v1-v4: Direct AsyncIO

Attempted to use asyncio directly with Kit's event loop. All approaches failed with AssertionError.

v5: nest_asyncio + Official API

Used nest_asyncio.apply() with Isaac Lab's official setup_async_generation() and env_loop(). First success!

v9-v12: subprocess Methods

Tried subprocess.Popen, IPython !, os.system. Output appeared in wrong cell due to Isaac Sim stdout hijacking.

v13-v17: SKIP_SIMULATION

Don't start Isaac Sim in Jupyter. Run standalone script via os.system(). Output displays correctly!

v18-v20: File Structure & Video

Reorganized output to hierarchical folders. Created encode_video_standalone.py for GPU video encoding.

v21-v23: Multi-Select UI & Timing

SelectMultiple widgets for batch processing. Added notebook execution time measurement.
